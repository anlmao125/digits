Digits App

Digits is a contact and note management web application built with Next.js 14. It showcases modern full-stack technologies and follows best practices for development with TypeScript, authentication, and database interaction.

Key Technologies

  • Next.js 14 with App Router for page routing and layout
  • PostgreSQL as the database, managed using Prisma ORM
  • NextAuth.js for secure authentication and user role support
  • React Hook Form for efficient and dynamic form management
  • Yup for schema-based form validation
  • Bootstrap 5 via react-bootstrap for a clean and responsive UI
  • ESLint for code quality enforcement

Getting Started

  1. Install PostgreSQL
    Ensure PostgreSQL is installed on your machine. Then, create a new database: 
    createdb digits
  2. Clone this repository
    Clone the Digits repository to your local machine: 
    git clone https://github.com/anlmao125/digits.git
  3. Install dependencies
    Navigate into the cloned directory and run: 
    npm install
  4. Set up environment variables
    Copy sample.env to .env and update the DATABASE_URL to point to your local PostgreSQL database: 
    cp sample.env .env
  5. Push Prisma schema and generate client
    This command will apply database migrations and generate Prisma client: 
    npx prisma migrate dev
  6. Seed the database
    Populate your database with default users and data: 
    npx prisma db seed
  7. Run the development server
    Start the app locally: 
    npm run dev
