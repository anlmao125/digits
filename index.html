Digits App

Digits is a Next.js 14 application built for managing user contacts and notes. It demonstrates:

  • Next.js App Router
  • PostgreSQL with Prisma
  • NextAuth.js for authentication and roles
  • Form handling with React Hook Form and validation via Yup
  • Bootstrap 5 UI using React-Bootstrap
  • ESLint integration

Installation

  1. Install PostgreSQL and create a database: 
    createdb digits
  2. Clone this repo: 
    git clone https://github.com/anlmao125/digits.git
  3. Install dependencies: 
    npm install
  4. Create a .env file from sample.env, and set DATABASE_URL
  5. Push Prisma schema: 
    npx prisma migrate dev
  6. Seed the database: 
    npx prisma db seed
  7. Run the development server: 
    npm run dev
